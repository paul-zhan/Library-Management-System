# 📚 Library Management System (GUI + SQLite3)

A Python-based Library Management System with a graphical user interface (GUI) and SQLite3 database integration. This system helps manage library operations like book search, borrow, reservation, and return through a user-friendly interface.

## 🛠️ Features

- GUI built with Python (Tkinter)
- SQLite3 for database management
- Search books by title, author, or ISBN
- Borrow and return books functionality
- Book reservation feature
- Input validation and error handling

## 💻 Technologies Used

- Python 3.x
- SQLite3
- Tkinter 
- Standard Python libraries
